# Solo Leveling

Application full-stack avec React et NestJS.

## Structure

- **client/** - Application React (Vite, TypeScript)
- **server/** - API NestJS (TypeScript)

## Installation

```bash
# Client
cd client && npm install

# Server
cd server && npm install
```

## Démarrage

```bash
# Développement avec Docker
docker-compose up
```

## Technologies

- Client: React, TypeScript, Tailwind CSS, Vite
- Server: NestJS, Prisma
- Database: PostgreSQL (via Docker)
